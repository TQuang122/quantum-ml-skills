# qml-paper-replication quickstart prompts

## Prompt 1 — turn a paper into an implementation plan

Use `qml-paper-replication` to extract the assumptions, missing details, baselines, and implementation plan from this QML paper. Produce a paper-to-code checklist instead of jumping straight into implementation.

## Prompt 2 — judge replication status honestly

Use `qml-paper-replication` to compare my implementation against this paper. Identify whether the result should be labeled `replicated`, `approximated`, or `unresolved`, and document all meaningful deviations.
