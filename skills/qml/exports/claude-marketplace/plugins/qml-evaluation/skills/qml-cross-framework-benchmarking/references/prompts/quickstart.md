# qml-cross-framework-benchmarking quickstart prompts

## Prompt 1 — design a fair comparison matrix

Design a controlled benchmark comparing PennyLane + PyTorch, Qiskit-backed execution, and any native Qiskit ML branch for this task. Keep dataset, metrics, seeds, and runtime accounting consistent, and report caveats explicitly.

## Prompt 2 — audit an existing benchmark

Audit this benchmark report for fairness. Check whether interfaces, backends, shots, compile costs, and preprocessing were controlled well enough to support the conclusions.
