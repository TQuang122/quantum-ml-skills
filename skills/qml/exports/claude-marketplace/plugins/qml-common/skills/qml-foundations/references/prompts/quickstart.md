# qml-foundations quickstart prompts

## Prompt 1 — scope a new QML task

Analyze this problem as a QML task. Decide whether it is a good fit for quantum machine learning, choose the most plausible modeling family, define at least one classical baseline, and state the metrics and backend assumptions before any implementation begins.

## Prompt 2 — research-to-plan

Read this idea or paper summary and turn it into a practical QML implementation plan. Identify the task type, likely dataset assumptions, variational vs kernel choice, evaluation risks, and the minimum baseline experiments needed.
