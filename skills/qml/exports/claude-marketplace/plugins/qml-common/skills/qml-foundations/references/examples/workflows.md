# qml-foundations example workflows

## Example 1 — decide if a classifier should be quantum

- Input: small binary classification task with 2-8 features
- Skill outcome: recommends a variational classifier only if a classical baseline and evaluation plan are defined first

## Example 2 — reject weak QML framing

- Input: vague request claiming “use quantum to improve accuracy”
- Skill outcome: forces a baseline, metric plan, and realistic simulator/backend choice before any circuit work starts
