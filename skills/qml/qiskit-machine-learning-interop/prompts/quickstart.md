# qiskit-machine-learning-interop quickstart prompts

## Prompt 1 — justify native Qiskit ML use

Assess whether this task actually needs native Qiskit Machine Learning abstractions such as `EstimatorQNN`, `SamplerQNN`, or `TorchConnector`, or whether plugin-backed PennyLane execution is sufficient.

## Prompt 2 — build a controlled interop branch

Create a minimal native Qiskit ML comparison branch for this experiment. Keep preprocessing and evaluation aligned with the PennyLane baseline and document why the branch exists.
