# Template: pennylane-qnn

Starter skeleton for a PennyLane-first variational classifier.

## Files

- `src/config.py` — model/device configuration
- `src/model.py` — device creation, encoding, variational block, QNode, and prediction helpers

## Use this template when

- you need a clean starting point for a variational classifier
- the main work is ansatz, encoding, measurement, or model modularity
