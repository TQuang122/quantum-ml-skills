# Template: qml-pytorch-training

Starter skeleton for a reusable PennyLane + PyTorch training workflow.

## Files

- `src/config.py` — training configuration
- `src/metrics.py` — accuracy helper
- `src/train.py` — optimizer setup, train/eval loops, and fit orchestration

## Use this template when

- the forward path already works
- the main need is a cleaner training loop with validation and logging
